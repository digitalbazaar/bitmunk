Settings

Directories

Download Directory Help

The directory your purchases will be stored in when complete.

Temporary Directory Help

The directory where temporary download files will be stored.

Purchasing

Max Download Rate Help

The maximum download rate in kilobytes per second (1024 KiB/s is equivalent to 1 MiB/s).

Max Excess Bandwidth Help

The maximum excess download bandwidth to allow before starting to download from additional sellers. Specified in kilobytes per second (1024 KiB/s is equivalent to 1 MiB/s).

Max Concurrent Pieces Help

The maximum number of file pieces to download at one time. This setting applies to each downloading file individually.

Selling

Max Upload Rate Help

The maximum upload rate in kilobytes per second (1024 KiB/s is equivalent to 1 MiB/s).

Auto-sell Purchases Help

When enabled your purchases will automatically be available for sale on the Bitmunk network using your default payment scheme or an automatically created scheme that you can edit.

Network Access

To sell items on the Bitmunk network other users must be able to connect to your software.

Universal Plug and Play Help

If you have an Internet connection through a gateway or firewall you may need to enable UPnP to allow users outside your local network to connect to you.

What is Universal Plug and Play?

External (router) Port Help

You can let Bitmunk pick a local port for you or choose one yourself with this option.

Warning: Changing this setting will force your software to immediately restart.

Logging

History Size Help

Adjust the number of messages stored in the log.

Main Log Level Help

The Main Log records the most recent information, warning and error messages reported by the client.

Console Log Level Help

The Console Log can be viewed with Firebug or any browser tool that supports the console.log function.

Level Description
None No logging will be done. Not Recommended.
Error Logs critical problems only.
Warning Logs Errors and Warnings about potential problems. Recommended level for normal use.
Info Logs Errors, Warnings and non-important information notices.
Debug Also log coarse internal messages. Recommended only for developers.
Verbose Also log detailed internal messages. Recommended only for developers.
Max Log everything. Recommended only for developers.

Resource Caching Help

When enabled the web browser will cache resources to make page rendering times quicker.
